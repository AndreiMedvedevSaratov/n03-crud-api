# n03-crud-api by Andrei Medvedev

## Routes
### GET      /api/users
### POST     /api/users
### GET      /api/users/id
### PUT      /api/users/id
### DELETE   /api/users/id

## Example :
### http://localhost:5000/api/users with "GET" - all users you will get


## Install
### npm install

## Development mode
### npm run start:dev

## Production mode
### npm run start:prod

## Horizontal scaling - multi mode
### npm run start:multi
